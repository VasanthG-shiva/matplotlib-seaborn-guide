01 — Overview

Two Libraries, One Goal

MPL

Matplotlib

The foundational Python plotting library. Gives you complete control over every element — axes, ticks, colors, labels, sizes. Verbose but infinitely flexible. Created by John Hunter in 2003.

Low-level Full control Any chart type Publication plots Embedding in GUIs
SNS

Seaborn

A high-level wrapper built on top of Matplotlib. Makes statistical charts beautiful and easy. Works natively with Pandas DataFrames. Less code, smarter defaults, built-in themes.

High-level Statistical DataFrame-native Beautiful defaults EDA / Analysis
# Installation pip install matplotlib seaborn # Standard imports used throughout this guide import matplotlib.pyplot as plt import seaborn as sns import pandas as pd import numpy as np sns.set_theme(style="darkgrid") # Apply Seaborn theme globally

02 — Chart Types

Every Chart, Explained

Bar Chart
Compares quantities across discrete categories using rectangular bars.
Matplotlib Seaborn Categorical comparison
# Matplotlib plt.bar(categories, values, color='steelblue') plt.show() # Seaborn sns.barplot(data=df, x='category', y='value')
Line Chart
Shows trends and changes over a continuous variable — most commonly time.
Matplotlib Seaborn Trends over time
# Matplotlib plt.plot(x, y, marker='o', linestyle='-') # Seaborn (with confidence interval) sns.lineplot(data=df, x='time', y='value', hue='group')
Scatter Plot
Reveals relationships, clusters, and correlations between two numeric variables.
Matplotlib Seaborn Correlation / Clusters
# Matplotlib plt.scatter(x, y, c=colors, s=sizes, alpha=0.7) # Seaborn (+ regression line option) sns.scatterplot(data=df, x='x', y='y', hue='category', size='value')
Histogram
Displays the distribution of a single numeric variable by grouping into bins.
Matplotlib Seaborn Distribution shape
# Matplotlib plt.hist(data, bins=30, edgecolor='white') # Seaborn (KDE overlay included) sns.histplot(data=df, x='value', kde=True, bins=30, hue='group')
Box Plot
Summarizes distribution using median, quartiles, and outliers across categories.
Matplotlib Seaborn Compare distributions
# Matplotlib plt.boxplot(data, labels=labels) # Seaborn (easier multi-group) sns.boxplot(data=df, x='category', y='value', hue='group', palette='Set2')
Violin Plot
Like a box plot but also shows the probability density — great for multimodal data.
Matplotlib Seaborn Distribution shape
# Matplotlib plt.violinplot(data, positions, showmedians=True) # Seaborn (much simpler) sns.violinplot(data=df, x='group', y='value', inner='box')
Heatmap
Visualizes a 2D matrix of values using color intensity — perfect for correlation matrices.
Seaborn Correlation matrix 2D grid data
# Seaborn (native, best choice) corr = df.corr() sns.heatmap(corr, annot=True, fmt=".2f", cmap='coolwarm', vmin=-1, vmax=1) # Matplotlib (manual) plt.imshow(matrix, cmap='viridis') plt.colorbar()
Pie / Donut Chart
Shows part-to-whole relationships. Use sparingly — bar charts are usually clearer.
Matplotlib Part-to-whole
# Matplotlib only (Seaborn lacks native pie) plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=90) # Donut: add a white center circle circle = plt.Circle((0,0), 0.5, color='white') plt.gca().add_patch(circle)
Pair Plot
Matrix of scatter plots for every pair of numeric variables. EDA powerhouse.
Seaborn Exploratory Analysis
# Seaborn — one line! sns.pairplot(df, hue='species', diag_kind='kde') # With regression lines sns.pairplot(df, kind='reg')
KDE Plot
Smooth, continuous estimate of a distribution — a polished alternative to histograms.
Seaborn Smooth distribution
# Seaborn sns.kdeplot(data=df, x='value', hue='group', fill=True) # 2D KDE (density contour map) sns.kdeplot(data=df, x='x', y='y', fill=True, cmap='Blues')
Regression Plot
Scatter plot with a fitted regression line and confidence interval band.
Seaborn Linear relationship
# Seaborn — includes CI band sns.regplot(data=df, x='x', y='y', ci=95, scatter_kws={'alpha':0.5}) # With faceting by category sns.lmplot(data=df, x='x', y='y', hue='group')
Stacked Bar Chart
Bar chart showing sub-group composition within each category simultaneously.
Matplotlib Seaborn Part-to-whole per group
# Matplotlib plt.bar(x, val1, label='A') plt.bar(x, val2, bottom=val1, label='B') # Seaborn via pandas + matplotlib df.plot(kind='bar', stacked=True)

03 — Data Type Guide

Which Chart for Which Data?

Every dataset has a data type combination on X and Y axes. Pick the right chart by matching your data types below.

Data Types: NUM — Numerical (continuous) CAT — Categorical (discrete) TIME — Time / Ordered sequence NONE — Single variable only
NUM × NUM Numerical vs Numerical

Two continuous numbers — find relationships, trends, clusters.

Scatter Show relationship & correlation between two numeric columns (e.g. height vs weight)
Regression Scatter + trend line — shows direction & strength of linear relationship
2D KDE Density of points in 2D space — great for large datasets with overlap
Heatmap Correlation matrix across many numeric columns at once (sns.heatmap on df.corr())
CAT × NUM Categorical vs Numerical

Compare a number across groups — the most common chart scenario.

Bar Compare aggregate values (mean/sum) across groups (e.g. avg salary per dept)
Box Plot Show spread, median, outliers per group — great for spotting skew
Violin Box plot + density curve — reveals if distribution is bimodal or skewed
Strip / Swarm Plot every data point per group — use for small datasets to avoid hiding data
CAT × CAT Categorical vs Categorical

Two category columns — compare proportions or counts across groups.

Grouped Bar Side-by-side bars for each sub-category (e.g. gender vs department count)
Stacked Bar Show composition — how sub-groups make up each main category
Pie / Donut Part-to-whole for a single category — keep to ≤5 slices for clarity
Heatmap (count) Pivot table as color grid — show frequency of every CAT×CAT combination
TIME × NUM Time vs Numerical

A number changing over time — show trends, cycles, seasonality.

Line Chart Best for continuous time series — shows trend, peaks, valleys (e.g. stock price)
Area Chart Line chart with filled area — emphasizes magnitude of change over time
Bar (time) Good for discrete time intervals (monthly, yearly) — easier to compare exact values
Multi-line Compare multiple groups over time (sns.lineplot with hue= parameter)
NONE × NUM Single Numerical Variable

Understand how one numeric column is distributed — shape, spread, skew.

Histogram See the shape of distribution — is it normal, skewed, bimodal?
KDE Plot Smooth version of histogram — easier to compare between groups
Box Plot Summary statistics at a glance — median, IQR, outliers instantly visible
Rug Plot Tick marks for every data point on an axis — shows exact data density
NUM × NUM ×N Many Numerical Variables

Explore relationships across ALL numeric columns at once — EDA mode.

Pair Plot Grid of scatter plots — every column vs every other. Best first step in EDA
Corr Heatmap Correlation matrix as a colored grid — quickly find which columns are related
Parallel Coords Compare multiple dimensions simultaneously for multi-class pattern detection
X Axis Y Axis Goal Best Chart(s)

CategoricalCategoricalShow composition / countStacked Bar, Grouped Bar, Pie

04 — Head to Head

Matplotlib vs Seaborn

Feature Matplotlib Seaborn
Level of abstraction Low — explicit control of every element High — handles layout automatically
Code verbosity More lines needed for same result Much less code, faster to prototype
Default aesthetics Plain, functional defaults Publication-ready themes built in
DataFrame support Manual — pass arrays/lists Native Pandas DataFrame integration
Statistical plots Manual computation required Built-in: CI, KDE, regression, etc.
Heatmap plt.imshow() — manual sns.heatmap() — one line
Pair plot Not available natively sns.pairplot() built-in
Pie chart plt.pie() native No native pie chart
3D plotting mpl_toolkits.mplot3d Not supported
Animation FuncAnimation supported Not supported
Customization Unlimited fine-grained control Good but limited deep customization
Learning curve Steeper — more concepts to learn Gentle — intuitive API
Used together ✓ Yes! sns plots are matplotlib objects — plt.title(), plt.savefig() work on both

05 — Decision Guide

When to Use Which

▸ Use Matplotlib when…

  • You need full pixel-level control over the chart
  • Creating animations or interactive plots
  • Embedding charts in desktop GUIs (Tkinter, Qt)
  • Building 3D visualizations
  • Making pie/donut charts
  • Your data isn't in a Pandas DataFrame
  • You need a chart type Seaborn doesn't support
  • Publication figures with custom layout

▸ Use Seaborn when…

  • Working with a Pandas DataFrame
  • Exploratory Data Analysis (EDA)
  • You want beautiful defaults quickly
  • Statistical visualizations (CI, KDE, regression)
  • Correlation heatmaps and pair plots
  • Comparing distributions across groups
  • Faceting / multi-panel plots by category
  • You want to write less code

💡 Pro Tip: Use Both Together

Seaborn is built on Matplotlib — every Seaborn plot is a Matplotlib figure. Create your chart in Seaborn for speed and beauty, then fine-tune with Matplotlib commands.

sns.heatmap(corr, annot=True) # Seaborn for the hard part plt.title("Correlation Matrix", fontsize=16) # Matplotlib to polish plt.tight_layout() plt.savefig("chart.png", dpi=300) # Matplotlib to export
